# ELE362 Workspace

This repository contains both projects used together:

- ELE362_Gamepad_Slave
- ELE362_Console_Master

## Open As Multi-Root Workspace

In VS Code, open the `ELE362.code-workspace` file to load both projects at once.

## Build / Upload

Each project keeps its own `platformio.ini` and can be built/uploaded independently with PlatformIO.
